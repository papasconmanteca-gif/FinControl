# FinControl

Aplicación web de administración financiera personal desarrollada con tecnologías web nativas. Permite registrar el sueldo mensual y los gastos individuales, y obtener automáticamente información financiera consolidada en tiempo real.

---

## Funcionalidades

- Ingreso y validación del sueldo mensual.
- Registro de gastos con fecha, descripción, categoría y monto.
- Clasificación de gastos en nueve categorías: Alimentación, Transporte, Vivienda, Salud, Entretenimiento, Educación, Ropa, Servicios y Otros.
- Historial de gastos en tabla con opción de eliminar cada registro.
- Cálculo automático del total gastado, dinero disponible y porcentaje del sueldo utilizado.
- Cálculo de promedio de gasto diario, semanal y mensual.
- Mensaje de estado financiero automático según el porcentaje utilizado (bueno, moderado o alto).
- Diseño responsivo adaptado a dispositivos móviles y de escritorio.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- Visual Studio Code

---

## Cómo ejecutar el proyecto

No se requiere instalación ni conexión a internet.

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en cualquier navegador moderno (Chrome, Firefox, Edge, Safari).
